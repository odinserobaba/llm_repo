### **1. Web Components с Shadow DOM**
**HTML:**
```html
<custom-element>
  <p slot="content">Текст в Shadow DOM</p>
</custom-element>
```
**Селекторы:**
- **CSS (внутри Shadow DOM):** `p[slot="content"]`
- **Playwright:** `page.locator('custom-element >>> p')` (устаревший синтаксис, используйте `::part`).
- **Shadow DOM API:** `document.querySelector('custom-element').shadowRoot.querySelector('p')`

- Для **Shadow DOM** используйте `::part` и `::slotted`.

# 35 примеров CSS-селекторов для разных движков

## 1. **Vue.js**

```html
<template>
  <div :class="{'active': isActive}" @click="handleClick">
    <span v-if="showText">{{ message }}</span>
    <ul>
      <li v-for="(item, index) in items" :key="item.id">
        {{ item.name }}
      </li>
    </ul>
  </div>
</template>
```

**CSS селекторы:**
```css
.active { background: #e3f2fd; }
[v-if] { color: #333; }
[v-for] { border: 1px solid #ddd; }
[\\:class] { padding: 10px; }
[\\@click] { cursor: pointer; }
[v-bind\\:key] { margin: 5px; }
```

## 2. **React (JSX)**

```jsx
function Component() {
  return (
    <div className={`box ${isActive ? 'active' : ''}`}>
      <button onClick={handleClick} style={{color: 'red'}}>
        Click me
      </button>
      {items.map(item => (
        <div key={item.id} data-testid="item">
          {item.name}
        </div>
      ))}
    </div>
  );
}
```

**CSS селекторы:**
```css
.box { border: 1px solid #ccc; }
.box.active { background: #4CAF50; }
[onClick] { cursor: pointer; }
[style] { font-weight: bold; }
[data-testid="item"] { padding: 5px; }
[className~="box"] { margin: 10px; }
```

## Классификация по движкам:

### **MVVM/Многофункциональные:**
1. Vue.js - `:class`, `@click`, `v-if`, `v-for`
2. Angular - `[property]`, `(event)`, `*ngIf`, `[(ngModel)]`
3. Ember.js - `{{action}}`, `{{if}}`, `{{each}}`
4. Knockout - `data-bind`, `click`
5. Aurelia - `repeat.for`, `click.trigger`

### **React-подобные:**
6. React - `onClick`, `className`, `data-testid`
7. Preact - `onclick`, `class`
8. Inferno - `onClick`, `className`
9. Nerv - `key`, `data-item`
10. SolidJS - `onClick`, `data-id`

### **Компилируемые:**
11. Svelte - `on:click`, `{#if}`, `{#each}`
12. Marko - `on-click()`, `for|item|`
13. Stencil - `:host`, `::slotted`
14. Polymer/Lit - `?active`, `@click`, `slot`

### **Минималистичные:**
15. Alpine.js - `x-data`, `@click`, `x-show`
16. Stimulus - `data-controller`, `data-action`
17. HTMX - `hx-get`, `hx-trigger`
18. Hotwire - `turbo-frame`, `data-controller`

### **Функциональные/Рективные:**
19. Mithril - `onclick`, `key`
20. Hyperapp - `onclick`, `class`
21. Choo - `onclick`, `emit`
22. VanJS - `onclick`, `key`

### **Специализированные:**
23. Qwik - `onClick$`, `class={{}}`
24. Dojo - `classes`, `key`
25. ReasonReact - `key`, `className`
26. Elm - `onClick`, `attribute`

### **Реактивные системы:**
27. S.js - `S.value`, `S.root`
28. Surplus - `S.data`, `onclick`
29. Cycle.js - `select`, `events`

### **MVC/Классические:**
30. Backbone - `<%= %>`, `<% %>`
31. Meteor - `{{#each}}`, `{{#if}}`

35. Bobril - `onClick`, `key`

Каждый движок имеет свои уникальные атрибуты и подходы к связыванию данных, что отражается в CSS-селекторах, которые нужно использовать для стилизации.
