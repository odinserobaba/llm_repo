# Few‑shot prompting: учим по примерам

**Идея:** даём 2–3 примера → модель повторяет стиль.

## Шаблон
```
Формат ответа: "Термин — короткое объяснение (1 строка)".

Примеры:
Кэш — память для повторных запросов.
Ретривер — поиск по базе по смыслу.

Теперь ответь в том же формате:
<список новых терминов>
```
