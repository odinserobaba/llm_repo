# ReAct: мышление + действия (через инструменты)

**Идея:** модель планирует шаги и вызывает инструменты.

## Шаблон
```
Ты агент. У тебя есть инструменты:
- search_web(query)
- get_doc(id)

Правила:
- если нужна информация, сначала вызови инструмент
- ответ давай только после инструментов

Вопрос: {question}
```
