# Zero‑shot prompting: как спросить без примеров

**Идея:** один чёткий запрос → один точный ответ.

## Шаблон
```
Ты эксперт по <тема>.
Задача: <что нужно получить>.
Ответ: коротко, по делу, 3–5 пунктов.
```

## Пример
```
Ты эксперт по кэшированию LLM.
Задача: объясни, зачем нужен кэш в продакшне.
Ответ: коротко, по делу, 3–5 пунктов.
```
