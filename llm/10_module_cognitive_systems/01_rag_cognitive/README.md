# Cognitive RAG — RAG с рассуждениями

```
┌─────────────────────────────────────────────┐
│  COGNITIVE RAG: библиотекарь, который       │
│  рассуждает вслух, а не просто приносит      │
├──────────┬──────────┬────────────────────────┤
│ Якорь   │ Механика │ Прорыв                 │
│ Библио- │ retrieve │ Multi-hop + CoT        │
│ текарь  │ → reason │ → точность ↑           │
│ -эксперт│ → generate│                        │
└──────────┴──────────┴────────────────────────┘
```

---

## 🎯 Якорь

Библиотекарь-эксперт не просто приносит страницу — он рассуждает: «сначала посмотрю раздел X, потом Y, затем сопоставлю». Cognitive RAG делает то же: цепочка мыслей или multi-hop retrieval.

---

## 📖 Термины

| Термин | Что это |
|--------|---------|
| **Reasoning RAG** | Ретривер → LLM «подумай» → следующий запрос → финальный ответ |
| **Self-RAG** | Модель сама решает: нужен ли retrieval, корректен ли ответ |
| **Multi-hop** | Несколько запросов к базе: вопрос A → промежуточный → вопрос B |
