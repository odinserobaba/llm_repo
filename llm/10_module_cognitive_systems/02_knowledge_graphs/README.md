# Knowledge Graphs — графы знаний

```
┌─────────────────────────────────────────────┐
│  KNOWLEDGE GRAPH: паутина фактов —          │
│  сущности связаны рёбрами                   │
├──────────┬──────────┬────────────────────────┤
│ Якорь   │ Механика │ Прорыв                 │
│ Паутина │ Узлы +   │ GraphRAG: поиск        │
│ фактов  │ рёбра    │ по связям, не только   │
│          │          │ по вектору             │
└──────────┴──────────┴────────────────────────┘
```

---

## 🎯 Якорь

Граф знаний — как паутина: узлы (Иван, Яндекс, Москва) связаны рёбрами (работает_в, находится_в). Поиск идёт не только «по смыслу», но и по структуре связей.

---

## 📖 Термины

| Термин | Что это |
|--------|---------|
| **Узел (Node)** | Сущность: персона, организация, локация |
| **Рёбро (Edge)** | Связь: works_at, located_in |
| **GraphRAG** | RAG, где retrieval использует граф (пути, соседи) |
| **networkx** | Библиотека Python для графов |
