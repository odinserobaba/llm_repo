# RAG prompting: ответ только по контексту

**Идея:** модель отвечает **только** на основе найденных документов.

## Шаблон
```
Используй КОНТЕКСТ ниже.
Если ответа нет — скажи "нет данных".

КОНТЕКСТ:
{context}

ВОПРОС:
{question}

Ответ: коротко, 1–3 предложения.
```

## Пример
```
Используй КОНТЕКСТ ниже.
Если ответа нет — скажи "нет данных".

КОНТЕКСТ:
Векторная БД хранит эмбеддинги и ищет похожие документы.

ВОПРОС:
Зачем нужна векторная база?

Ответ: коротко, 1–3 предложения.
```
