# Reflexion: самокоррекция

**Идея:** сначала ответ, потом самопроверка и улучшение.

## Шаблон
```
Сначала дай ответ.
Потом проверь себя по 3 пунктам: точность, полнота, краткость.
После проверки дай улучшенный ответ.
```
