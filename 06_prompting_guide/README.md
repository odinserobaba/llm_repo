# Prompting Guide — примеры

Это отдельная папка с практическими примерами техник из Prompt Engineering Guide.  
Источник: https://www.promptingguide.ai/  

## Что внутри
- `promt.md` — ваш системный промпт (копия `promt.md` 1–309)
- `01_zero_shot.md` — zero‑shot
- `02_few_shot.md` — few‑shot
- `03_chain_of_thought.md` — контролируемые рассуждения (без вывода скрытых мыслей)
- `04_prompt_chaining.md` — цепочки промптов
- `05_rag_prompt.md` — RAG‑инструкция
- `06_react.md` — ReAct‑подход (мышление + действие через инструменты)
- `07_reflexion.md` — рефлексия/самокоррекция

## Как использовать
1) Открой файл с техникой.  
2) Скопируй шаблон промпта.  
3) Подставь свою задачу.  

Если нужно — добавлю больше примеров под твои задачи.
