# Контролируемые рассуждения (без раскрытия цепочки мыслей)

**Идея:** просим думать шагами, но **выводить только финальный ответ**.

## Шаблон
```
Реши задачу. Подумай пошагово, но выведи только финальный ответ.
Ответ: <коротко>.
```

## Пример
```
Реши задачу. Подумай пошагово, но выведи только финальный ответ.
Сколько токенов примерно уйдёт на текст из 1200 слов?
Ответ: коротко, одной строкой.
```
