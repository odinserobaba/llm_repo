### **1. Web Components с Shadow DOM**
**HTML:**
```html
<custom-element>
  <p slot="content">Текст в Shadow DOM</p>
</custom-element>
```
**Селекторы:**
- **CSS (внутри Shadow DOM):** `p[slot="content"]`
- **Playwright:** `page.locator('custom-element >>> p')` (устаревший синтаксис, используйте `::part`).
- **Shadow DOM API:** `document.querySelector('custom-element').shadowRoot.querySelector('p')`

---

### **Дополнительные советы:**
- Для **Shadow DOM** используйте `::part` и `::slotted`.
- Для **динамических атрибутов** (например, `data-*`) используйте `attributeContains` в XPath или CSS.
- Для **тестирования анимаций** в Playwright используйте `page.waitForFunction`.
