Вот 10 примеров в формате Markdown, где приведён HTML-код и различные варианты селекторов для работы с ним (включая CSS, XPath, Playwright, BeautifulSoup и другие подходы):

```markdown
# Примеры HTML-кода и селекторов

## 1. Простой параграф
**HTML:**
```html
<p class="text">Привет, мир!</p>
```

**Селекторы:**
- CSS: `.text`
- XPath: `//p[@class='text']`
- Playwright: `page.locator('.text')`
- BeautifulSoup: `soup.find('p', class_='text')`

---

## 2. Список с элементами
**HTML:**
```html
<ul id="menu">
  <li class="item active">Главная</li>
  <li class="item">О нас</li>
</ul>
```

**Селекторы:**
- CSS: `#menu .item.active`
- XPath: `//ul[@id='menu']/li[contains(@class, 'active')]`
- Playwright: `page.locator('#menu .item.active')`
- BeautifulSoup: `soup.find('ul', id='menu').find('li', class_='active')`

---

## 3. Форма с полями
**HTML:**
```html
<form>
  <input type="text" name="username" placeholder="Имя пользователя">
  <input type="password" name="password" placeholder="Пароль">
</form>
```

**Селекторы:**
- CSS: `input[name="username"]`
- XPath: `//input[@name='username']`
- Playwright: `page.locator('input[name="username"]')`
- BeautifulSoup: `soup.find('input', attrs={'name': 'username'})`

---

## 4. Таблица с данными
**HTML:**
```html
<table>
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
  </tr>
  <tr>
    <td>Алиса</td>
    <td>25</td>
  </tr>
</table>
```

**Селекторы:**
- CSS: `table tr:nth-child(2) td:first-child`
- XPath: `//table/tr[2]/td[1]`
- Playwright: `page.locator('table tr:nth-child(2) td:first-child')`
- BeautifulSoup: `soup.find('table').find_all('tr')[1].find_all('td')[0]`

---

## 5. Кнопка с атрибутами
**HTML:**
```html
<button id="submit" data-action="save" disabled>Сохранить</button>
```

**Селекторы:**
- CSS: `button#submit[data-action="save"]`
- XPath: `//button[@id='submit' and @data-action='save']`
- Playwright: `page.locator('button#submit[data-action="save"]')`
- BeautifulSoup: `soup.find('button', id='submit', attrs={'data-action': 'save'})`

---

## 6. Вложенные div-элементы

- BeautifulSoup: `soup.find('div', class_='container').find('div', class_='header')`

### Советы для Playwright и TestBed:
- **Playwright**: Используйте `locator.filter()` для сложных условий (например, `page.locator('li').filter({ hasText: 'Тест' })`).
- **TestBed**: Для динамических данных используйте `fixture.whenStable()` перед проверками.
- **Angular**: Для тестирования сервисов используйте `TestBed.inject(YourService)`.
