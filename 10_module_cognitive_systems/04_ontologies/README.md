---

## 🎯 Якорь

Онтология — каталог: «Персона» имеет свойства «имя», «должность»; «Организация» — «название», «ИНН». Поиск идёт по структуре: «все персоны, связанные с организацией X».

---

## 📖 Термины

| Термин | Что это |
|--------|---------|
| **Класс** | Категория сущности: Person, Organization |
| **Свойство** | Атрибут: name, birthDate |
| **Ontology-based Search** | Фильтрация по классам и связям |
| **Schema** | Описание схемы в JSON-LD, OWL |
