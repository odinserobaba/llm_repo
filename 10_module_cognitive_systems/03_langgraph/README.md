# LangGraph — оркестрация LLM

```
┌─────────────────────────────────────────────┐
│  LANGGRAPH: карта маршрутов — не линейка,   │
│  а лабиринт с поворотами и циклами          │
├──────────┬──────────┬────────────────────────┤
│ Якорь   │ Механика │ Прорыв                 │
│ Карта   │ StateGraph│ Циклы, ветвления,     │
│ маршрутов│ узлы+рёбра│ human-in-the-loop     │
└──────────┴──────────┴────────────────────────┘
```

---

## 🎯 Якорь

LangGraph — не просто цепочка A→B→C. Это граф состояний: можно вернуться назад, разветвиться, вставить «стоп и спроси человека». Как карта с перекрёстками.

---

## 📖 Термины

| Термин | Что это |
|--------|---------|
| **StateGraph** | Граф: узлы = функции, рёбра = переходы |
| **add_node** | Добавить шаг (retrieve, generate) |
| **add_edge** | Связь: «после A иди в B» |
| **add_conditional_edge** | Ветвление по условию |
