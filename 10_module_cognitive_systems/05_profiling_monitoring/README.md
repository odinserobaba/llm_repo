---

## 🎯 Якорь

Профилирование и мониторинг — приборная панель: каждый вызов LLM, время, токены, стоимость. Evaluators оценивают качество ответов автоматически.

---

## 📖 Термины

| Термин | Что это |
|--------|---------|
| **Trace** | Один полный запуск pipeline |
| **Span** | Отдельная операция (retrieval, generation) |
| **Evaluator** | Функция оценки: relevance, length |
| **Langfuse / LangSmith** | Сервисы для трассировки и метрик |
